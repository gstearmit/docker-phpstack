# Docker for PHP Stack

## Soft

- PHP 5.5 / 5.6  
  - fpm, redis, mongo, gd, imagick, xdebug, memcache, memcached, icu (55.1), opcache
- Ruby
- Git
- Nodejs + NPM
- Composer

## Install

```shell
# PHP 5.5
docker pull rkit/docker-phpstack:5.5
# PHP 5.6
docker pull rkit/docker-phpstack:5.6
```
