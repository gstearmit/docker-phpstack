# Docker for PHP Stack

PHP 5.5, 5.6, 7.1

## Extensions

- gd
- opcache
- zip
- pdo_mysql
- mysqli
- mbstring
- redis
- mongo
- memcached
- xdebug
- imagick
- intl

## Soft

- Composer
- Node.js + NPM
- Git

## Install

```shell
# or
docker pull rkit/docker-phpstack:5.5
# or
docker pull rkit/docker-phpstack:5.6
# or
docker pull rkit/docker-phpstack:7.1
```
