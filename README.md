# Docker for PHP Stack

PHP 5.5, 5.6, 7.1

## Extensions

- gd
- opcache
- zip
- pdo_mysql
- mysqli
- mbstring
- redis
- mongo
- memcached
- xdebug
- imagick
- intl

## Soft

- Ruby
- Git
- Nodejs + NPM
- Composer

## Install

```shell
# PHP 5.5
docker pull rkit/docker-phpstack:5.5
# PHP 5.6
docker pull rkit/docker-phpstack:5.6
# PHP 7.1
docker pull rkit/docker-phpstack:7.1
```
