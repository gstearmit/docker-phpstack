# Docker for PHP Development Stack

## Soft

- PHP (5.5, 5.6)  
  fpm, redis, mongo, gd, imagick, xdebug, memcache, memcached, icu (55.1), opcache
- Ruby
- Git
- Nodejs + NPM
- Composer

## Install

```shell
# with php 5.5
docker pull rkit/docker-php:5.5
# with php 5.6
docker pull rkit/docker-php:5.6
```
